# Machine Learning Based Traffic Signal Optimization 

## Project Overview
This project predicts travel time using machine learning models (Linear Regression, SVM, MLP) and performs signal phase optimization.

## Dataset
- cleansed_traffic_monitoring.csv
- Description: traffic conditions + signal phase + travel time

## Project Structure
- data/: datasets
- notebooks/: model training and optimization
- figures/: visualization
- src/: optional scripts
